# DirectoryLister-themes

These are the themes I've created/modified for
[DirectoryLister](https://github.com/DirectoryLister/DirectoryLister).

### Available themes

|name|description|

files.lawniczak.me


## Installation

Copy the whole directory into `resources/themes` in the main DirectoryLister
catalog. Then change the theme in `resources/config.php`:

    <?php
    return array(
        // Basic settings
        // ...
        'theme_name'                => 'NAME-OF-THEME',
        //...
    );
    

## Usage

After installation it's done ;) Just refresh the page.
